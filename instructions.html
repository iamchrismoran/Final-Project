Welcome to the Pixel Art Creator

If you have time and patience, I hope you can create some interesting art of even some pixel art for game design. Most edits you make get autosaved and the Gallery page should show all of your previous works, and allow you to load them back up and keep working on them.
How to use the features
Changes the size of each block. Updating this will not autosave. Handy if you want need larger or smaller blocks to easy the design or to fill up space according to your browser window.
These update the number of blocks horizontally (width) and vertically (height) in real time. Each time you make a change, it will autosave your design. Your design will stay in place according to its leftmost and topmost positions. If you make it wider or taller, it will add blanks blocks. If you make it narrower or shorter and there are colored blocks that get cut out, they are lost blocks.
Here's the color picker. I hope it works well enough on all browsers. You can select colors from the gamut or use the dropper to pick some color anywher else on your screen. If you click on a block that has the exact same color you have selected, it will "erase" it (paint it white). Every time you click on a block to change the color, your art is autosaved.
Name your artwork. When you add/change the title of your work, it will autosave. If you change the name without making changed to the art, it will save it over the other title. If you want to restart, click the "Reset the design" button.
This will pop open a version of the design with pixel blocks. It will stay up until you hide it again and it updates in real time when you make changes.
This clears the colors, ands resets it back to all white.
When you print using the system print or button (if available), it should open up a print dialog that only shows the artwork - suitable for the fridge.
